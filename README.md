# Prescription System

Digital prescription management system connecting doctors and pharmacies with electronic prescribing, medication tracking, and drug interaction alerts.

## Features

- Electronic prescription creation and management
- Drug interaction checking
- Pharmacy-side prescription fulfillment tracking
- Patient medication history
- Prescription templates for common conditions

## Tech Stack

Dart, Flutter, Firebase, Cloud Firestore

## Getting Started

### Prerequisites

- Flutter SDK
- flutter

### Installation

```bash
git clone https://github.com/karthik-idikuda/Prescription-System.git
cd Prescription-System
flutter pub get
```

### Usage

```bash
flutter run
```
